# Bewise — тестовое задание (Python backend)

Источник: `Hexlet/ru-test-assignments/backend/Python/Bewise/test_python.md`

## Задачи

1. С помощью Docker (предпочтительно `docker-compose`) развернуть образ с СУБД (предпочтительно — PostgreSQL). Обеспечить сохранность данных при рестарте контейнера (volume).
2. Реализовать на Python 3 простой веб‑сервис (FastAPI или Flask), который:
   - Принимает `POST` запросы с JSON вида `{"questions_num": integer}`.
   - После получения запроса запрашивает с публичного API викторины `https://jservice.io/api/random?count=1` указанное количество вопросов.
   - Сохраняет вопросы в БД. Минимум: `id` вопроса, текст вопроса, текст ответа, дата создания вопроса.
   - Если вопрос уже есть в БД — делает дополнительные запросы, пока не получит уникальный вопрос.
   - Возвращает **предыдущий сохранённый** вопрос. Если его нет — пустой объект.
3. Предоставить инструкции по сборке/запуску docker‑образа сервиса, настройке и пример запроса.
