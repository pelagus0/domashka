# AppBooster — backend test assignment

Источник: `Hexlet/ru-test-assignments/backend/Python/AppBooster/backend.md`

## Кратко

Нужно реализовать сервис A/B‑экспериментов с единственным API‑эндпойнтом:
мобильное приложение присылает `Device-Token` и получает список экспериментов
с назначенными вариантами (опциями), при этом:

1. Девайс всегда должен попадать в одну и ту же группу (стабильное распределение).
2. Эксперимент действует только для “новых” девайсов: если эксперимент создан после первого запроса от девайса, девайс не должен о нём узнать.
3. Добавить 2 эксперимента:
   - `button_color` c равными долями по 33.3%
   - `price` с долями 75%/10%/5%/10%
4. Сделать страницу/эндпойнт статистики: список экспериментов и распределение девайсов по опциям.
