# Ivelum — Hacker News proxy

Источник: `Hexlet/ru-test-assignments/backend/Python/Ivelum/python.md`

## Задача

Сделать локальный HTTP‑прокси для страниц Hacker News (`https://news.ycombinator.com`):

- Прокси должен модифицировать текст на страницах: **после каждого слова из 6 букв** добавлять знак `™`.
- Навигация по ссылкам HN должна оставаться внутри прокси (браузер не уходит на `news.ycombinator.com`).

Технологии: последняя версия Python, можно любые публичные библиотеки, стиль PEP8.
