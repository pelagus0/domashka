# UpTrader — Django app “древовидное меню”

Источник: `Hexlet/ru-test-assignments/backend/Python/UpTrader/test_python.md`

## Требования из задания

1. Меню реализовано через template tag.
2. Всё, что над выделенным пунктом — развернуто. Первый уровень вложенности под выделенным пунктом тоже развернут.
3. Хранится в БД.
4. Редактируется в стандартной админке Django.
5. Активный пункт меню определяется по URL текущей страницы.
6. На одной странице может быть несколько меню — определяется по названию.
7. По клику — переход по заданному URL (явный URL или named url).
8. На отрисовку каждого меню — ровно **1 запрос к БД**.
