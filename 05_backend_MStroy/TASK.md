# ООО «МСтрой» — TreeStore

Источник: `Hexlet/ru-test-assignments/backend/ООО МСтрой/README.md`

## Задача

Дан массив объектов с полями `id` и `parent` (и произвольными полями).
Нужно реализовать класс `TreeStore`, который принимает массив в конструктор и
реализует методы:

- `getAll()`
- `getItem(id)`
- `getChildren(id)`
- `getAllParents(id)` — цепочка родителей к корню (порядок важен)

Требование: **максимальное быстродействие**, минимум обходов массива.

В этом решении задача обёрнута в HTTP API (FastAPI), а элементы хранятся в PostgreSQL.
